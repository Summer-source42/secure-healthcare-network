# Resume Bullets

## Project Title

**Secure Healthcare Enterprise Network Design**  
Cisco Packet Tracer | Cisco IOS | ASA Firewall | OSPF | HSRP | VLANs | Wireless | VoIP

## Best Resume Version

- Designed and implemented a secure multi-floor healthcare enterprise network supporting LAN, WLAN, VoIP, DMZ, ISP, and simulated cloud connectivity.
- Configured VLAN segmentation, inter-VLAN routing, OSPF, HSRP gateway redundancy, LACP EtherChannel, centralized DHCP/DNS, and WLC-managed wireless access.
- Implemented Cisco ASA firewall zones, NAT/PAT, DMZ access control, SSH-restricted management, and ACL-based traffic filtering between internal, DMZ, and outside networks.
- Documented topology, IP addressing, device configurations, verification commands, failover testing, and troubleshooting notes for a production-style network portfolio.

## Short LinkedIn / GitHub Description

Built a secure enterprise healthcare network simulation using Cisco Packet Tracer. The project includes VLAN segmentation, inter-VLAN routing, OSPF, HSRP, EtherChannel, Cisco ASA firewall zones, NAT, DMZ services, DHCP/DNS, WLC-managed wireless, VoIP, and documented validation/testing evidence.
